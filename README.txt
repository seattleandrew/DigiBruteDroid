DigiBruteDroid is a 4-Digit PIN (now supporting 5 digits) Brute Force attack for USB-OTG Android devices (such as the Samsung Galaxy S3). 
This code was written to support DigiStump's DigiSpark Model A and the DigiKeyboard Library.
This code was written by Seattleandrew and is intended for Mobile Device Security Research.
Improvements and expansions to this code are welcomed and appreciated.
